# Manhattan Associates — Topic 2 Prospect Walkthrough

Static HTML slide deck for the Manhattan Associates Topic 2 account POV walkthrough.

## Run locally

```bash
open index.html
```

Or serve with any static server:

```bash
python3 -m http.server 8768
```

Then visit `http://localhost:8768`.

**Navigation:** `←` / `→`, Space, Page Up/Down, click left/right thirds of the slide, or progress dots. Swipe on mobile.

## Deploy to Vercel

```bash
npx vercel
```
